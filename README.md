# Trabalho de Engenharia da Computação

## 📋 Descrição
Este repositório contém exercícios práticos de JavaScript, desenvolvidos como material de estudo para conceitos fundamentais de programação.

## 📁 Arquivos JavaScript

### 1. **helloWord.js**
- **Propósito**: Programa introdutório que demonstra o primeiro contato com JavaScript
- **Conteúdo**: Imprime "Hello, World!!" no console
- **Conceito**: Saída de dados com `console.log()`
- **Como executar**: `node helloWord.js`

### 2. **Variáveis.js**
- **Propósito**: Demonstra os tipos de declaração de variáveis em JavaScript
- **Conteúdo**:
  - Variáveis `var`: declaração com ou sem valor inicial
  - Variáveis `let`: declaração com ou sem valor inicial
  - Variáveis `const`: declaração obrigatória com valor e imutáveis
- **Exemplos**: 
  - `var nome = 'José'`
  - `let sobrenome = 'Silva'`
  - `const PI = 3.14159`
- **Como executar**: `node Variáveis.js`

### 3. **forLoop.js**
- **Propósito**: Demonstra o funcionamento de loops `for` em JavaScript
- **Conteúdo**: Quatro exemplos práticos de iteração:
  1. Contagem crescente de 0 a 10
  2. Contagem regressiva de 10 a 0
  3. Iteração sobre elementos de um array (frutas)
  4. Impressão de tabuada (10 × 10)
- **Conceitos**: Inicialização, condição, incremento/decremento, acesso a índices
- **Como executar**: `node forLoop.js`

### 4. **whileLoop.js**
- **Propósito**: Demonstra o funcionamento de loops `while` em JavaScript
- **Conteúdo**: Quatro exemplos práticos de iteração condicional:
  1. Contagem crescente de 0 a 10
  2. Contagem regressiva de 10 a 0
  3. Iteração sobre array de nomes (Alice, Bob, Charlie)
  4. Impressão de tabuada (10 × 10)
- **Conceitos**: Condição de continuação, incremento manual, controle de fluxo
- **Como executar**: `node whileLoop.js`

## 🚀 Como Utilizar

Para executar qualquer arquivo JavaScript, use o Node.js:

```bash
node <nome-do-arquivo>.js
```

Exemplo:
```bash
node helloWord.js
node Variáveis.js
node forLoop.js
node whileLoop.js
```
